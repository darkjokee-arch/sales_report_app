통합 영업망 사용 꿀팁!

1. 접속 보안 및 접속 방법

새로운 통합 PIN 번호는 7890 입니다.
접속 주소를 스마트폰 웹 브라우저의 '홈 화면에 추가' 기능으로 바탕화면에 꺼내두시면 앱처럼 편하게 쓰실 수 있습니다!

2. 로드맵 (지도) 200% 활용하기

상단의 지도 로드맵 탭을 누르면 모든 영업건이 지도에 핀으로 표시됩니다.

  • 핀을 누르면 건물 세부 정보가 뜹니다.
  • 주소 텍스트를 누르면 카카오맵 길안내, 상세보기 안의 카카오/티맵 버튼을 누르면 설치된 내비게이션으로 바로 연결됩니다!

3. 타사검토 상태 즉각 공유!

영업건 상태를 타사검토요청(토스)로 변경하고 저장하시면, 자동으로 전체 채팅방에 알림 봇(Bot)이 발생하여 4개사 모두가 즉시 확인할 수 있습니다.

4. 실시간 커뮤니티 채팅

화면 우측 하단의 버튼을 눌러보세요!

4개사가 함께 실시간으로 업무 소통을 나눌 수 있습니다. 닉네임(예: 세일(홍길동))을 설정하면 바로 대화가 가능합니다!
